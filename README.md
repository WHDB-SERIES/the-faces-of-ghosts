# the-faces-of-ghosts
entry of the WHDB series; classic faces, because of our saudade.
___

the faces of ghosts is the first entry of WHDB, aimed to overwrite Roblox's migration from Classic Faces to Dynamic Heads, by forcing all dynamic faces to be deleted and replaced with their classic counterpart.
this is COMMUNITY-DRIVEN; while I have provided a basic filter-list, it is by no means exhaustive. sustain the project by forking the basic list and adding extra id's.
